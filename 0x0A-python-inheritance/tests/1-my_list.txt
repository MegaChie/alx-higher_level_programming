Using ``MyList``
-----------------

First import user defined function:
	>>> MyList = __import__('1-my_list').MyList

case one:
	>>> my_list.print_sorted()
	[-1, 0, 3, 4]

Case two:
	>>> my_list.print_sorted()
	[-1, 0, 3, 4]

case three:
	>>> my_list = MyList()
	>>> my_list.append(999999999999)
	>>> my_list.append(-1)
	>>> my_list.append(0)
	>>> print(my_list)
	[999999999999, -1, 0]

case four:
	>>> my_list.print_sorted()
	[-1, 0, 999999999999]
